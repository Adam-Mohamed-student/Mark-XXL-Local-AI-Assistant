# Mark-XXL-Local-AI-Assistant
MARK XXL is a privacy-first, local voice AI assistant (J.A.R.V.I.S) running fully on-device: offline STT (Whisper/Vosk) → local LLM via Ollama with streaming/tool-calling → TTS (EdgeTTS/Kokoro/ElevenLabs). It ships with a PyQt HUD plus tools to control apps, browser, files, and OS automation.
